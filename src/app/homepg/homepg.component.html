Welcome to my World News Aggregator!

This app uses the World News API (by skycraft LLC) on APILayer

REQUIREMENTS -

Acquire and set up an API key:

  1. Make an account at the API Layer website.
  2. Go to your profile and get an API key.
  3. Create a file

    apiKeys.ts

    in the

    src

    directory of this application.
  4. Save this file with the contents export const apilayer_key = "yourAPIkey";
